# Onboarding Steps

## Canonical Onboarding Workflow

This is the ONLY supported onboarding process. No deviations or customizations.

### Step 1: Account Setup (Day 1)
- Account creation
- Initial user invitations
- Basic configuration

### Step 2: First Win (Days 2-7)
- Complete first production job (ShopFlow)
- OR post first RFQ (ShopLink)
- OR run first calculation (ShopCalc Pro)
- **This step is REQUIRED before proceeding**

### Step 3: Team Onboarding (Days 8-14)
- Invite team members
- Set up workflows
- Configure notifications

### Step 4: Integration (Days 15-21) - Core/Premium Only
- API access setup
- Webhook configuration
- Data export setup

### Step 5: Optimization (Days 22-30)
- Review usage patterns
- Optimize workflows
- Advanced feature training

## First Win Requirements

The "First Win" must be completed before moving to Step 3. This ensures users understand core value.

### ShopFlow First Win
- Create a production job
- Add at least one routing step
- Mark job as complete

### ShopLink First Win
- Post an RFQ
- Receive at least one bid
- Review bid details

### ShopCalc Pro First Win
- Run a calculation
- Save results
- Export or share results

## What's NOT Included

- Custom onboarding paths
- Industry-specific training
- Custom workflow design
- Extended onboarding timelines
- On-site training

## Support During Onboarding

- Email support available
- Documentation and guides
- Video tutorials
- Standard onboarding checklist

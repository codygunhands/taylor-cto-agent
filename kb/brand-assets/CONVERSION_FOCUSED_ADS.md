# Conversion-Focused Ad Templates

## Philosophy

**Keep it simple. Focus on conversions. Don't waste Dustin's time.**

These templates are based on proven ad formats that convert. No fancy graphics, no overthinking - just clear messaging that drives action.

## High-Converting Ad Formats

### 1. LinkedIn/Facebook Ad (1200x628px)

**Why it works**: Standard format, proven CTR
**Format**: Headline + Benefit + CTA

**Template**: `linkedin-ad`

**Structure**:
- **Headline**: Bold, attention-grabbing (56px, Inter Bold)
- **Benefit**: Clear value prop (32px, Inter Regular)
- **CTA**: Action button (28px, Inter SemiBold)

**Example**:
```
Headline: "Stop Losing Deals to Competitors"
Benefit: "ShopWare helps you price accurately and win more bids"
CTA: "Start Free Trial"
```

### 2. Google Display Ad (300x250px)

**Why it works**: Standard Google size, high visibility
**Format**: Short headline + CTA

**Template**: `google-display`

**Structure**:
- **Headline**: Concise, benefit-focused (24px, Inter Bold)
- **CTA**: Clear action (18px, Inter SemiBold)

**Example**:
```
Headline: "Price Jobs Faster"
CTA: "Try ShopWare Free"
```

### 3. Square Ad (1080x1080px)

**Why it works**: Instagram/Facebook feed format, visual-first
**Format**: Large headline + Benefit + CTA

**Template**: `square-ad`

**Structure**:
- **Headline**: Large, bold (72px, Inter Bold)
- **Benefit**: Supporting text (36px, Inter Regular)
- **CTA**: Prominent button (32px, Inter SemiBold)

**Example**:
```
Headline: "Win More Deals"
Benefit: "Accurate pricing in seconds"
CTA: "Get Started"
```

## Conversion Principles

### 1. **Clear Value Proposition**
- What problem does it solve?
- What benefit does the customer get?
- Why should they care?

### 2. **Action-Oriented CTA**
- Use action verbs: "Start", "Get", "Try", "Learn"
- Be specific: "Start Free Trial" not "Click Here"
- Create urgency: "Get Started Today"

### 3. **Benefit-Focused Headlines**
- Focus on outcomes, not features
- Use numbers when possible: "Save 10 Hours/Week"
- Address pain points: "Stop Losing Deals"

### 4. **Keep It Simple**
- One main message per ad
- Don't overcrowd with text
- Let white space work for you

## What NOT to Do

❌ **Don't**: Use multiple fonts or colors
❌ **Don't**: Overcomplicate the message
❌ **Don't**: Use vague CTAs like "Learn More"
❌ **Don't**: Include too much text
❌ **Don't**: Use low-contrast colors

✅ **Do**: Use Inter font (brand standard)
✅ **Do**: Keep it grayscale (brand standard)
✅ **Do**: Focus on one clear message
✅ **Do**: Use high-contrast (black on white)
✅ **Do**: Make CTA obvious and actionable

## Testing

**Always test**:
- Different headlines
- Different CTAs
- Different benefits
- A/B test variations

**Track**:
- Click-through rate (CTR)
- Conversion rate
- Cost per acquisition (CPA)
- Return on ad spend (ROAS)

**Keep it simple. Focus on conversions. Don't waste time.**

These templates are proven formats that work. Use them, test variations, and optimize based on data.
